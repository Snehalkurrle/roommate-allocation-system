# Roommate Allocation System

## Project Overview
The Roommate Allocation System is a web-based application developed as a final year engineering project.
It helps users find and allocate suitable roommates based on preferences, availability, and location.

## Features
- User registration and login
- Roommate matching based on preferences
- Room / flat listings
- Search and filtering functionality
- Simple and user-friendly interface

## Tech Stack
- Frontend: HTML, CSS, JavaScript / React
- Backend: Node.js / Express
- Database: MySQL / MongoDB
